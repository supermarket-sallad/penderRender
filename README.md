# penderRender

-----

#### A deferred renderer in TouchDesigner, heavily stylized. 

* check out the example file first

-----

penderRender is a deferred renderer in touchDesigner

it was created in order to create highly stylized visuals - primarilly as a toonshader - but it grew in scope.

It is still being develloped as I use it - but shared under MIT license to allow others to use it. Some jank is to be expected.
